# Лабораторная работа №7

В проект добавлено локальное развёртывание через Docker Compose.

## Что вынесено в сервисы

1. `web` — основной Django-сайт.
2. `notification-service` — отдельный микросервис уведомлений. Django отправляет ему JSON, а сервис уже отвечает за отправку сообщения в Telegram.

Так проект не остаётся полностью монолитным: отправка уведомлений вынесена из Django-приложения в отдельный контейнер.

## Как запустить

Из папки `config`, где лежит `manage.py`, выполнить:

```bash
docker compose up --build
```

После запуска сайт будет доступен по адресу:

```text
http://127.0.0.1:8000/
```

Если Telegram-токен не задан, проект всё равно запускается, просто уведомления в Telegram отправляться не будут.

## Проверка

```bash
docker compose exec web python manage.py test
```
