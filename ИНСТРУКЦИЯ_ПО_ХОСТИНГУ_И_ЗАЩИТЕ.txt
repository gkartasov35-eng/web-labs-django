Внутри папки config добавлены файлы:

1. README_HOSTING_RENDER.md — инструкция по выкладке проекта на Render.
2. DEFENSE_INFO.md — готовая информация для защиты лабораторных 5, 6 и 7.
3. render.yaml — конфигурация для автоматического деплоя.
4. .env.example — пример переменных окружения.

Важно: настоящий публичный URL появляется только после подключения репозитория к аккаунту Render/Replit/PythonAnywhere.
Точный URL нужно взять из панели хостинга после деплоя.

CI/CD:
В проект добавлен GitHub Actions workflow .github/workflows/ci-cd.yml.
Он запускает проверки Django, тесты, миграции и сборку Docker-образа.
Для автоматического деплоя на Render нужно добавить GitHub Secret RENDER_DEPLOY_HOOK_URL.
Если секрет не задан, CI всё равно работает, а CD-шаг пропускается без ошибки.
